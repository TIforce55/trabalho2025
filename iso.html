Características da ISO/IEC 25010

Funcionalidade Avalia se o software realiza corretamente as funções esperadas. Subcaracterísticas: Adequação Funcional: O software faz o que é necessário? Precisão Funcional: Os resultados são corretos e exatos? Adequação Funcional para Tarefas: É eficiente para os objetivos dos usuários?

Confiabilidade Avalia se o software funciona corretamente por longos períodos sem falhas. Subcaracterísticas: Maturidade: Quantas falhas ocorrem? Disponibilidade: Está disponível quando necessário? Tolerância a Falhas: Se algo dá errado, o sistema continua funcionando? Recuperabilidade: Pode se recuperar rapidamente de falhas?

Usabilidade Avalia se o software funciona corretamente por longos períodos sem falhas. Subcaracterísticas: Maturidade: Quantas falhas ocorrem? Disponibilidade: Está disponível quando necessário? Tolerância a Falhas: Se algo dá errado, o sistema continua funcionando? Recuperabilidade: Pode se recuperar rapidamente de falhas?

Eficiência Avalia o uso de recursos do sistema e a velocidade de resposta. Subcaracterísticas: Comportamento em Tempo de Resposta: Tempo que leva para reagir. Utilização de Recursos: Usa pouca memória, CPU, etc.? Capacidade (Throughput): Quantidade de trabalho que consegue processar.
